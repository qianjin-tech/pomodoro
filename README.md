# 番茄钟

一个简洁的桌面番茄钟应用，基于 Electron 构建。

## 功能

- **工作 / 短休 / 长休** 三种计时模式，4 个番茄后自动进入长休
- **圆形进度环** 直观展示剩余时间
- **桌面通知** 计时结束时弹出系统提醒
- **音效提示** 三声提示音
- **系统托盘** 最小化到托盘，右键菜单支持置顶
- **可自定义** 工作时长、休息时长、长休间隔

## 启动方式

- **双击** `start-pomodoro.bat`
- **命令行** `npm start`
- **浏览器** 直接打开 `pomodoro.html`

## 快捷键

| 按键 | 操作 |
|------|------|
| 空格 | 开始 / 暂停 |

## 技术栈

- Electron
- HTML + CSS + JavaScript（零前端依赖）
- Web Audio API 生成提示音
- Web Notification API 桌面通知
